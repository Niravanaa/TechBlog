Personal Project: Projectile Motion Simulator

One of my most recent personal projects is a Python application for simulating 2D projectile motion and visualizing trajectories. With this project, I aimed to:

  • Simulate 2D projectile motion.
  • Visualize trajectories using Matplotlib.
  • Calculate and display key metrics such as max height, range, and time of flight.
  • Provide options to adjust x-axis and y-axis ranges.
  • Allow users to toggle grid lines on the plot.
  • Clear the plot for a fresh start.

This project has been a fantastic opportunity to apply my programming skills to real-world physics simulations. Plus, it's been a great learning experience in GUI development with Tkinter. You can view the repository using this link.
